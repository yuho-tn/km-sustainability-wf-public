Part 2 — Adoption Criteria

採用基準：使う条件 / 使わない条件（WHEN / WHEN NOT）

よく使われる5モジュールの採用条件と非採用条件を1枚にまとめる。クライアント説明・自己レビュー両用。プレビューはモジュール一覧と同一描画。

8-4 / 8-1 アコーディオン .wf-acc-h3 / .wf-acc-h2
CLOSED
取締役会での承認事項
OPEN（デフォルト）
取締役会での承認事項
2008年：長期環境目標 / 2020年：5マテリアリティ...
WHEN — こんな時に使う
  • H3配下に同列項目が3つ以上（TCFD開示4要素／DE&I 3テーマ）
  • 各項目が3段落以上の長文セクション
  • すべてを必読としなくてよい（選択的に読ませたい）
  • ページ構造をコンパクトに見せたい
WHEN NOT — 使わない場面
  • 項目が1〜2個しかない（直接表示で十分）
  • ページの主目的そのものを表す内容（畳むと意図が消える）
  • 既存原文が短いリード文＋数パラグラフのみ（盛らない）
  • 用語1個だけの補足 → モーダル（9-2）に
代替：用語=9-2モーダル／並列見せ=7-3カード／単発本文=直接表示 初期状態：<details open> 必須 階層：H3直下=.wf-acc-h3 / H4直下=.wf-acc-h4
